原生js异步树
=================================================================
### 配置项列表:
| option | description |
| :------: | :------: |
| "id" | 放置组件容器元素的id |
| "data" | 树形结构的数据 |
| "itemKey" | 树组件每一项的展示key与data中每一项的key相对照 |
| "isExpand" | 初始化是否展开全部节点 |
| "loadNode" | 加载节点的回调函数 |
